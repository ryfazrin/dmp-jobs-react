# HOW TO RUN

Run 
```
pnpm i

pnpm run dev
```

if command show 
```
➜  Local:   http://127.0.0.1:5173/ // or anything
➜  Network: use --host to expose
➜  press h to show help
```
Vite is run.

## Page
```
- Login Page
  url: BASE_URL/login
  
- Home Page 
  url: BASE_URL/
  1. Show All jobs
  2. Search
  3. Paging

- Detail Page
  url: BASE_URL/jobs/ID
  1. Show Detail Jobs
```
